In this second tutorial related to Node.js, we are going to set up a simple project that allows us to create a server with Node.js and run our code.

If you want to know, understand and master Node.js, it is good to start with plain Node.js, instead of starting with a framework like Express or Nest - we will start to study and master them once our knowledge of Node.js will be strong enough. So, in the following tutorials, we are going to understand pure Node.js discovering how powerful and fast it is.

  1. Download Node.js and install Node.js
  2. Create a project with Node.js

Download Node.js and install Node.js

The first step is to install Node.js on your computer.

You can download Node.js from the official website and install it.

It is strongly recommended to download the LTS version of Node.js, that is the stable version - Long-Term Support version.

 

Once you have installed Node.js, check if that works properly opening the terminal and running the following command:

 


node -v

The above command should return the version of Node.js that you have installed.

 


v18.16.0

If you run the above command and you have an error, it is possible that you did not install Node.js in the right way. Often, a common error is related to the PATH variables, in that case it is possible that your terminal is displaying this error message:

 


'node' is not recognized as an internal or external command, operable program or batch file.

If you do not have any errors, you can go ahead with this tutorial.

 

Once you have installed Node.js, you can start to interact with it by the terminal of your device.

Open your terminal and run the command node + enter. At this point you will be able to use Node.js by the terminal of your computer as you can see in the following example.

 

 

Now, let’s create a project with Node.js.

 

Create a project with Node.js

You have downloaded and installed Node.js. Let start the first project creating a empty folder whose name is nodeProject

 

Inside the directory nodeProject, create JavaScript files whose name is app.js - this file can have any name, however we usually set up the server in files with name app.js or server.js

 

Open app.js with Visual Studio Code or with the code editor that you prefer and add the following code:

 

.env

const http = require('http');

const server = http.createServer((req, res) => {
    res.writeHead(200, {'Content-Type': 'text/html'});
    res.end('Hello Node');
})

server.listen(3000);

Open the terminal in the root directory of the nodeProject and run the following command:

  • node app.js

 

At this point, you are running your first Node.js server and you should able to see the output of your code at the following url:

  • http://localhost:3000/

 

 

That’s great, at this point your computer is working as a server thanks to Node.js.

Don’t worry if you do not understand the above code 100%, we will explain it and other great functionalities of Node.js in the following tutorials.
