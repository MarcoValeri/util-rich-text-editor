Let's set up Go and create our first web app with it, a Hello World web server in Go.

 

As the first step, you will need to download and install Go on your device.

The official documentation has all the instructions for downloading and installing Go on Linux, Mac, and Windows.

 

I strongly recommend installing Go for the best development experience.

However, if you would like to try out Go code without installing anything, you can use The Go Playground, a tool created by Go team, lets you run Go code directly in your browser.

 

Set up a new project in Go

If you have installed Go, create a new folder where we will store the files for this first app. For this project, I will call the folder:

go-hello-world

 

Inside this folder, create the main file:

main.go

 

Let’s write our first and classic “hello message” in Go:

main.go

package main

import "fmt"

func main() {
    fmt.Println("Hello World")
}

Please, do not worry if you do not understand the code 100%, it will be clear following the next tutorials.

 

At this point, open the terminal and navigate to the folder go-hello-world.

 

Once you are at this location with your terminal, run the following command:

  • go run main.go

Terminal

$ go run main.go
Hello World

Well done, you have run your first Go program.
