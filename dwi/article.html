React: what is it?

React: what is it? React is a declarative, flexible, and open-source JavaScript library that is used to create user interfaces based on components. It can be used to develop single-page, server-rendered applications with its framework like Next.js and mobile app thanks React Native.

 

In this tutorial of DevWithIt we are going to be focusing only on React. However, once you know the basics of React, it will be easy to switch to React Native or to one of the React frameworks like Next.js, Remix or Gatsby.

 

You can use React to create an entire web app like we will do in this tutorial dedicated to React, as well as you can add React to an existing project - e.g in a PHP project, you can add some React components to a small part of the web app.

 

React was developed by Jordan Walke, one of Facebook’s software engineers. It was released in 2013 and today it is maintained by Meta.

 

After its first release, React became one of the most popular JavaScript libraries. Why? Let’s find out in the next paragraph when and why it is important to use React.

 

Why use React?

Why use React? Some advantages of React are:

  • It allows to build web apps in an efficient way

  • It allows to organise and maintain the code in an good way

  • It allows to reuse its components

  • It is declarative

  • It is composable

  • It is fast

  • It is efficient

  • It is SEO friendly

  • It guarantees stable code

 

Don’t worry if you do not understand all the above points, you will master all of them coding along this tutorial of DevWithIt where we will create a To Do App with React.

 

There are different reasons, the first that we have to mention is that this JavaScript library allows us to build complex UI in an efficient way, organising and maintaining the code in an efficient way.

 

In React everything is a component. Components are independent and reusable parts of code that can be reused throughout the application.

 

React is declarative, that means that it updates and renders components when data changes, as well as, it is composable, that means that the web app is composed of freely interchangeable components.

 

React is SEO friendly, fast, efficient and it guarantees stable code thanks to the maintenance of Meta and thanks to its strong community.

 

One of the main concepts and one of the performance benefits of React is the Virtual DOM, a lightweight JavaScript representation of the DOM - Document Object Model, used in declarative programming like React, VueJS and Angular. Basically, React uses the virtual DOM to track changes to the user interface and it updates the real DOM when necessary, improving the performance of the web apps by preventing unnecessary DOM manipulations.

 

React and JSX

If you want to be a React Engineer, you will use JSX quite often. JSX is an extension for JavaScript that allows you to write HTML markup inside a JavaScript file. As highlighted in the official React documentation, writing markup with JSX is what most React developers prefer to do.
