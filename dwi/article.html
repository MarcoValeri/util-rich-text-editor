How to create a responsive navbar with React and SCSS? It is possible to create a responsive nav menu without icons and without any JavaScript events like we do in this new tutorial of DevWithIt.

Why this tutorial? Well, creating navigation menu is often a tricky task and at DevWithIt we like the idea of creating a base code that you can use and customise through your projects. However, the latter is not the only reason. Navigation menu is one of the first components that is loaded by the browser so it is good practice to have a light nav menu, avoiding unnecessary JavaScript.

In fact, in this tutorial we are going to create a navigation menu with React and SCSS, that means that the burger menu will work perfectly only thanks to the CSS.

 

What do you need for following this tutorial? You need to have installed Node on your device and a code editor like Visual Studio Code.

In the first paragraph of this tutorial we set up the project and in the second one we create the navigation menu. If you are in a rush, feel free to go to the last paragraph of the tutorial where you will find the code of this guide in a GitHub repository.

Set up the project

As a first step, we have to create a new React project. At the moment, one of the best options is Vite so we are going to use it.

In your device, open a terminal and browse to the folder where you want to create this new project, then run the following command:

  • npm create vite

 

After this command, you have to enter in your terminal some information like:

  • The name of the project - I used react-responsive-navbar

  • Select the framework - we use React for this project

  • Select a variant - we use Vanilla JavaScript for this tutorial

 

At this point, we have to open the project with Visual Studio Code or in any other code editor, you can use the following commands by your terminal:

  • cd react-responsive-navbar

  • code .

 

Once the project is opened in Visual Studio Code, we have to install all the required packages and then run the project in the local server. You can do it by opening a new terminal in Visual Studio Code and then running these commands:

  • npm install

  • npm run dev

 

There we go, our project is running in the local environment.

 

Let’s now clean up our project, open the file src/index.css and remove all the code. Do the same for the file the file src/App.css and src/App.jsx

 

It is time to create App component in App.jsx, open this file and add the following code:

    return (

            <h1>Hello React</h1>
        </>

It is time to install SASS and to create some style rules that will help us to keep our navigation responsive and our code more readable. Open the terminal in Visual Studio Code and install sass with the following command:

  • npm install sass

 

Now, inside the folder assets you have to create two sub folders: scss and abstract. Inside the folder abstract we are going to create the file _mixins.scss where we create style rules that can be reused, for this project a mixin that allows us to work through the breakpoints. In this tutorial we use only the breakpoint @include('md') but we like the idea to keep the other ones, so you can customise your navigation menu.

 

src/assets/scss/abstract/_mixins.scss


/**
* Create a mixin that works for
* media queries in the following breakpoints:
*           xs
*           sm
*           md
*           lg
*           xl
*           xxl
*
* This project follow the logic of responsive-design
*/
@mixin respond($breakpoint) {
    @if $breakpoint == xs {
        @media only screen and (max-width: 380px) { @content };
    }
    @if $breakpoint == sm {
        @media only screen and (max-width: 575px) { @content };
    }
    @if $breakpoint == md {
        @media only screen and (max-width: 767px) { @content };
    }
    @if $breakpoint == lg {
        @media only screen and (max-width: 991px) { @content };
    }
    @if $breakpoint == xl {
        @media only screen and (max-width: 1199px) { @content };
    }
    @if $breakpoint == xxl {
        @media only screen and (max-width: 1499px) { @content };
    }
}

The setup of our project is ready, it is time to create the navigation menu.

 

Create nav component

As you know, in React everything is a component, so let’s create our component navbar. First, we are going to create the folder components inside the folder src. Then, we will create the folder of our component, its JSX file as well as its SCSS file. This is the approach that we use at DevWithIt when we create a component with React but feel free to use your way too.

 

src/components/Navbar/Navbar.jsx


import './Navbar.scss';

const Navbar = () => {
    return(
        <nav className="navbar">
            <h1>This is the Navbar component</h1>
        </nav>
    )
}

export default Navbar;

 

src/components/navbar/Navbar.scss


.navbar {
    border: 1px solid #f00;
}

We have now to import our Navbar component to App.jsx and the main.scss file as you can see in the following example.

 

src/App.jsx


import Navbar from "./components/navbar/navbar";

import './assets/scss/main.scss';

const App = () => {
    return (
        <>
            <Navbar />
        </>
    )
}

export default App;

Now, if you run your project again with npm run dev, you should see our navigation. Well, it is time to create our navigation. In this part of this guide of DevWithIt, I provide you with a basic responsive navigation menu. You can then use this code and customise for your personal projects.

 

src/components/Navbar/Navbar.jsx


import { Link } from 'react-router-dom';

import './Navbar.scss';

const Navbar = () => {
    return (
        <nav className='navbar'>
            <div className="navbar__toggle">
                <label htmlFor="menu" aria-label="Main Navigation Menu"></label>
                <input id="menu" className="navbar__input" name="menu" type="checkbox" aria-label="Main Navigation Menu" />
                <span className="navbar__span navbar__span--empty"></span>
                <span className="navbar__span"></span>
                <span className="navbar__span"></span>
                <ul className="navbar__menu list-menu">
                    <li className="navbar__container-logo"><h2><Link className="navbar__logo link-no-style h1" to="/">React Responsive Navbar</Link></h2></li>
                    <li className="navbar__menu-item"><Link className="body-md link-no-style" to="/">Home</Link></li>
                    <li className="navbar__menu-item"><Link className="body-md link-no-style" to="/about">About</Link></li>
                    <li className="navbar__menu-item"><Link className="body-md link-no-style" to="/portfolio">Portfolio</Link></li>
                    <li className="navbar__menu-item"><Link className="body-md link-no-style" to="/contact">Contact</Link></li>
                </ul>
            </div>
        </nav>
    )
}

export default Navbar;
