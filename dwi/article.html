In this new guide of Dev With It we are going to convert a jpg, png or other image formats to webp.

Nowadays, it is important to have a fast and responsive website and webp format allows us to use images with smaller size compared to jpg and png. If you used to inspect your website with Lighthouse, you probably saw the message Serve images in modern formats. That is what we will do in this new tutorial of Dev With It.

 

What is webp?

WebP is an image format for the web that allows to create images 25-34% smaller compared to other formats like jpg, png, jpeg, GIF, APNG and so on. WebP was created by Google, it was announced in September 2010 and it was released in April 2018. Today, it is supported by all modern browsers like Google Chrome, Microsoft Edge, FireFox, Opera, Safari and many more - you can see the full list in the official documentation of the web browsers natively support WebP.

 

How to convert an image to webp format

There are different ways to convert an image to webp format. In this tutorial we are going to use the cwebp conversion tool that is available for Windows, macOS and Linux.

You can install cwebp by choco for Windows, brew for macOS or red hat for Linux.

 

Windows Terminal

            
                choco install webp

 

 

macOS Terminal

            
                brew install webp

 

 

Linux Terminal

            
                yum install libwebp-tools

 

In this tutorial we will use choco for Windows but the commands are the same for macOS and Linux too.

 

Open Windows terminal and run the following command:

  • choco install webp

 

Now we are ready to convert jpg, png, jpeg, GIF, APNG and other image formats to webp. Create a “Webp Image” folder in your device, e.g., inside the document folder - you can name this folder as you prefer, and you can save this folder anywhere in your device.

 

At this point, save into “Webp Image” (see example below) an image that you would like to convert into webp. In this case, I will save a london.jpg image into this folder, I got it by Unplash.

 

 

Now, you have to open your terminal in this folder and then you have to run the following command:

  • cwebp -q 75 london.jpg -o london.webp

 

 

The above command takes a jpg image and converts it to webp. The number 75 is the default lossy quality setting, however you can set the value when you run the above command in the terminal - e.g., cwebp -q 50 london.jpg -o london.webp

 

 

That’s it, you have created your first webp image and now you are able to convert all images to webp, noting how your web page will get faster.

 

As you can see, cwebp created new .webp images keeping the original one. At this point, feel free to open both images to see that there is no difference between them. In the above example, the london.jpg image was 4586 KB compared to the 2473 KB to the london.webp format. Can you imagine how it can speed up your web pages?

 

In this tutorial we saw a simple and easy way to transform an image to webp format. However, there are other and different ways to achieve this goal like Node npm or other platforms. Feel free to use the way that suits you.
