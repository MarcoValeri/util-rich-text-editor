In Go programming languages, variables store data values. Just like in any other programming language, a variable acts as a named storage location in memory that can hold a specific value.

 

Golang allows you to declare variables in different ways.

 

Unlike some languages, Go needs you to specify the data type (like int, string, etc.) for each variable you create:

main.go

package main

func main() {
    var productID int = 1
    var productName string = "Go book"
    var productPrice float64 = 10.99
}

var keyword is used to create the variables of a particular type. If type is omitted, Go will get the data type by its value

main.go

package main

func main() {
    var website = "devwithgo"
}

In the above case, the variable website has a string data type.

 

With the var keyword, it is possible to create multiple variables at once:

main.go

package main

func main() {
    var userOne, userTwo = "Mark", "Alex"
    var name, age = "Mark", 40
}

The above code demonstrates declaring multiple variables with var. Here, both userOne and userTwo have the data type string. Likewise, name is a string, while age is declared as an integer (int).

 

In Go, it is possible to declare a variable without explicitly initializing it. In this case, the variable receives a default value specific to its data type:

In the above case, the variable website has a string data type.

 

With the var keyword, it is possible to create multiple variables at once:

main.go

package main

func main() {
    var productID int  // The default value is 0
    var productName string  // The default value is an empty string
    var productPrice float64  // The default value is 0.0
    var productAvailable bool // The default value is false
}

When you declare a variable in Go without explicitly assigning a value, it automatically receives a default value depending on its data type. These defaults include:

  • Integers: 0
  • Strings: empty string ("")
  • Floating-point numbers (float64): 0.0
  • Booleans: false

 

In Go, you can declare variables using the shorthand := operator. This is a common way to declare and initialize a variable in a single line. The data type of the variable is inferred from the assigned value

main.go

package main

func main() {
    productName := "Go book"
}

In the above example, the variable productName has a string data type.

 

With the shorthand :=, it is possible to declare a variable only inside the function, but you can not use it outside the function.

Indeed, you can use the var keyword for declaring a variable outside the function.

main.go

package main

var productId int

func main() {
    productId = 1
    productName := "Go book"
}

The fmt package in Go is a standard library package that provides functions for formatted input and output operations. This means it allows you to format and print the values of variables, along with other data, to the console or other output streams:

main.go

package main

import "fmt"

func main() {
    productID := 1
    productName := "Go book"
    productPrice := 10.99
    productAvailable := true

    fmt.Println("Product details:")
    fmt.Println("ID:", productID)
    fmt.Println("name:", productName)
    fmt.Println("price:", productPrice)
    fmt.Println("available:", productAvailable)
}

Terminal

$ go run main.go
Product details:
ID: 1
name: Go book
price: 10.99
available: true

Go enforces type safety, meaning once a variable is initialized with a specific data type, you can only assign new values that are compatible with that type. This helps prevent errors caused by accidentally assigning incompatible data types:

main.go

package main

import "fmt"

func main() {
    productName := "Go book"
    fmt.Println(productName)

    productName = "Go book new edition"
    fmt.Println(productName)
}

Terminal

$ go run main.go
Go book
Go book new edition

Go is a statically typed language. This means the data type of a variable is determined during its declaration, and it cannot be changed later:

main.go

package main

import "fmt"

func main() {
    productID := 1
    productID = "1"
    fmt.Println(productID)
}

Terminal

$ go run main.go
./main.go:7:14: cannot use "1" (untyped string constant) as int value in assignment
