Go is a statically typed programming language, meaning that a variable's type is determined at compile time and remains fixed.

This makes understanding Go data types crucial for writing correct and efficient code.

As we've seen, variables in Go always have a data type, either explicitly declared or inferred from their initial value.

 

Let's delve deeper into this topic with some practical examples.

 

Go data types

Go offers a variety of data types, we will discover all of them throughout the following guides. Let’s start to say that the data types in Golang are divided in four main categories:

  1. Basic type: numbers, strings, and Booleans
  2. Aggregate type: array and structs
  3. Reference type: pointers, slices, maps, functions, and channels
  4. Interface type

 

In this first part, we will focus on the basic types. These are the fundamental building blocks of Go programs, and you might have already encountered them in simple code example.

 

The three basic types in Go are:

  • Booleans
  • String
  • Numbers

 

Let’s dive into each of these types and understand how they work in Go.

 

Go booleans

Booleans are a fundamental data type in Go, representing logical values that can be either true or false.

You can declare a boolean variable using the bool keyword.

In Go, the zero value of a boolean is false, meaning that if you declare a boolean variable without assigning it an initial value, it will automatically be set to false. Booleans are essential for controlling program flow and making decisions based on conditions.

main.go

package main

import "fmt"

func main() {
    thisIsTrue := true
    thisIsFalse := true
    var thisIsFalseToo bool

    fmt.Println(thisIsTrue)
    fmt.Println(thisIsFalse)
    fmt.Println(thisIsFalseToo)
}

 

terminal

$ go run main.go
true
false
false

 

Go strings

 

Strings are a fundamental data type in Go, representing sequences of characters.

They are used to store and manipulate text-based information.

You declare a string variable using the string keyword.

In Go, the zero value of a string is an empty string (""), which is automatically assigned if you don't provide an initial value.

Strings are essential for a wide range of tasks, including storing user input, displaying messages, and passing data between different parts of an application.

 

main.go

package main

import "fmt"

func main() {
    var name string = "Marco" // String of 5 characters
    surname := "Valeri" // String of 6 characters
    var website string // String of 0 characters, empty string
}

 

You can use the built-in function len() to get the length of a string in Golang.

 

main.go

package main

import "fmt"

func main() {
    var name string = "Marco"
    surname := "Valeri"
    var website string

    fmt.Println(len(name))
    fmt.Println(len(surname))
    fmt.Println(len(website))
}

 

terminal

$ go run main.go
5
6
0

 

Go offers several ways to concatenate (join together) strings.

The most common and straightforward method is using the + operator.

Alternatively, the fmt package provides flexible formatting options, allowing you to include variables (even those of non-string types) within a string by using placeholders.

 

main.go

package main

import "fmt"

func main() {
    var name string = "Marco"
    surname := "Valeri"
    userName := userName + " " + surname
    age := 40
    var website string
    website := "devwithgo.dev"

    fmt.Println("Username:", userName)
    fmt.Println("Age:", age)
    fmt.Println("Website:", website)
}

 

terminal

$ go run main.go
Username: Marco Valeri
Age: 40
Website: devwithgo.dev

 

Go numbers

In Go, the numeric data types are divided into three main categories:

  1. Integers: Whole numbers without fractional components (e.g., 5, -12, 1000)
  2. Floating-Point Numbers: Numbers with decimal points or exponents (e.g., 3.14, -0.5, 6.02e23)
  3. Complex Numbers: Numbers with both real and imaginary parts (e.g., 2 + 3i)

 

Go integers number

In Go, integers are a fundamental data type representing whole numbers without fractional components.

You declare an integer variable using the int keyword, although Go provides several variations of integer types (e.g., int8, int16, int32, int64, uint8, etc.) to control their size and whether they can be negative.

In Go, the default value (zero value) of an integer is 0, which is automatically assigned if you don't provide an initial value.

Integers are essential for a wide range of tasks, including counting iterations, tracking scores, and performing arithmetic operations within your programs.

 

main.go

package main

import "fmt"

func main() {
    var userOneID int = 1
    userTwoID := 2
    var userThreeID int
}

 

terminal

$ go run main.go
1
2
0

 

Go supports both signed and unsigned integers, each available in four different sizes to balance memory usage and range of values:

 

Signed Integers:

  • int8: 8-bit signed integer (-128 to 127)
  • int16: 16-bit signed integer (-32768 to 32767)
  • int32: 32-bit signed integer (-2147483648 to 2147483647)
  • int64: 64-bit signed integer (-9223372036854775808 to 9223372036854775807)

 

Unsigned Integers:

  • uint8: 8-bit unsigned integer (0 to 255)
  • uint16: 16-bit unsigned integer (0 to 65535)
  • uint32: 32-bit unsigned integer (0 to 4294967295)
  • uint64: 64-bit unsigned integer (0 to 18446744073709551615)

 

Floating numbers

In Go, floating-point numbers are a fundamental data type used to represent numbers with fractional components, such as decimals or scientific notation.

You declare a floating-point variable using either the float32 or float64 keyword, depending on the desired precision.

The float32 type provides single-precision floating-point numbers, while the float64 type offers double precision.

In Go, the default value (zero value) for a floating-point number is 0.0, which is automatically assigned if you don't provide an initial value.

Floating-point numbers are essential for a wide range of applications, including scientific calculations, financial modelling, and graphics programming.

 

main.go

package main

import "fmt"

func main() {
    var productOne float64 = 9.99
    productTwo := 7.47
    var productThree float64

    fmt.Println(productOne)
    fmt.Println(productTwo)
    fmt.Println(productThree)
}

 

terminal

$ go run main.go
9.99
7.47
0

 

Go provides two floating-point types:

  • float32: IEEE-754 32-bit floating-point numbers
  • float64: IEEE-754 64-bit floating-point numbers

 

Go complex numbers

In Go, complex numbers are represented by two distinct types:

  • complex64: Complex numbers with float32 real and imaginary parts
  • complex128: Complex numbers with float64 real and imaginary parts

 

You can declare complex number variables using the complex64 or complex128 keywords. Go provides several built-in functions for working with complex numbers, including:

  • complex(r, i): Constructs a complex number from real (r) and imaginary (i) components
  • real(c): Extracts the real part of a complex number (c)
  • imag(c): Extracts the imaginary part of a complex number (c)

 

main.go

package main

import "fmt"

func main() {
    var complexNumberOne complex128 = complex(10, 2)
    var complexNumberTwo complex64 = complex(6, 3)
    var complexNumberThree complex128

    fmt.Println(complexNumberOne)
    fmt.Println(complexNumberTwo)
    fmt.Println(complexNumberThree)
}

 

terminal

$ go run main.go
(10+2i)
(6+3i)
(0+0i)
